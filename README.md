# 2024 Robot Code

### Controller Mapping
- Bumpers:
    - LT: Lower left arm
    - RT: Lower right arm
    - RB: Outtake
- Joysticks:
    - Left stick: Drive
    - Right stick: Rotate
- D-Pad:
    - Up: Climber arms up
    - Down: Climber arms down
- Buttons:
    - Y: Lock/Unlock wheels
    - X: Toggle field relative
    - A: Shoot note
    - B: Toggle intake
